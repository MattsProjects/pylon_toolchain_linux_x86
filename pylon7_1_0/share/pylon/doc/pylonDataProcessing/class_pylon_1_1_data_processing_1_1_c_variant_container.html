Detailed Description

CVariantContainer is a map-like container providing an interface like a C++ standard library.

Values can be processed like this:

CVariantContainer container;
for (const auto& element : container)
{
String_t name = element.first;
CVariant value = element.second;
}
Thread Safety:
The CVariantContainer class is not thread-safe.
