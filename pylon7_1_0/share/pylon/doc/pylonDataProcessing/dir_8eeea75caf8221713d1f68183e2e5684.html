file  PylonDataProcessingIncludes.h
 This file can be used to include all data processing classes.

 Contains the recipe class that can be used to load and run data processing designs.

 Contains the CUpdate class that can be used to check which output data has been generated by which CRecipe data update operation and what the corresponding input data is.
 
file  Variant.h
 Contains the variant class that can be used to represent any data type processed by a CRecipe.
 
file  VariantContainer.h
 Container for pushing data to inputs and receiving data from outputs.
 
file  VariantDataType.h
 Contains an enumeration that lists the built-in variant data types.
