pylon C++ Data Processing Programmer's Guide

Introduction

The Basler pylon Data Processing C++ API extends the pylon C++ image acquisition API to enable you to use vTools in your application code. vTools are the building blocks for creating data processing designs using the Workbench of the pylon Viewer. Data processing designs are saved in recipe documents. Prior to using the pylon Data Processing C++ API, you need to create a data processing design and save it as a recipe document in the Workbench. A recipe document contains information about the vTools used, information about how the vTools are connected, and further settings. More information about creating recipe documents can be found here.

Programmer's Guide The pylon Programmer's Guide is a brief guide to developing applications using the Basler pylon Data Processing C++ API. It can be used together with the pylon data processing sample code to get started.
API Reference The API Reference provides documentation about the Basler pylon C++ interface.
Sample Code The pylon sample code illustrates common use cases when working with the pylon C++ API.

Precautions

Caution: The code snippets provided in this guide are included as sample code only. Inappropriate code may cause your vTools to function differently than expected and may compromise your application. To ensure that the snippets will work properly in your application, you must adjust them to meet your specific needs and must test them thoroughly prior to use.
