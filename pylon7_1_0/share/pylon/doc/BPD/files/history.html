History#

This topic provides information about the current and previous versions of the Basler Product Documentation.

  • Added the pylon API Documentation section. The section replaces the following documents that were previously delivered with the Basler pylon Camera Software Suite:
    • C Programmer's Guide and Reference Documentation
    • C++ Programmer's Guide and Reference Documentation

  • Updated the documentation to reflect version 7.1 of the pylon Camera Software Suite. The following topics have been added:
    • Overview of the pylon GigE Configurator
    • pylon GigE Configurator (CLI Version)

  • Added information about the following new boost camera models:
    • boA1936-400cc

    • boA2448-250cm

    • boA2832-190cm

    • boA4096-180cm

  • Added missing exposure start delay values in the Specifics table of the Acquisition Timing Information topic for the ace 2 camera series.
  • Added missing exposure start delay values in the Specifics table of the Acquisition Timing Information topic for the ace 2 camera series.

  • Added the transfer bit depth as a frame rate-limiting factor to the Resulting Frame Rate topic.
  • Added information about connections in the Recipe Management topic.
  • Added a note about support for 5GigE and 10GigE network adapters in the Bandwidth Manager topic.
  • Added a note in the Recording topic about restrictions when using the feature for high-end applications and with the Compression Beyond feature. Also added advice how to work with cameras with high resolutions.

  • Corrected the supported barcode types in the Barcode Reader vTool topic.
